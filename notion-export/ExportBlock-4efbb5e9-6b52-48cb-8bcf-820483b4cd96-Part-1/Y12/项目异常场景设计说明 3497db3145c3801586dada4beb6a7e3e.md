# 项目异常场景设计说明

## 说明用于统一异常场景下的页面表现，避免出现：

- 空白页面
- 模块塌陷
- 破图
- 无反馈点击
- 状态不一致

UI 设计时，需要为每个核心模块预留异常态、空态、加载失败态和禁用态。

---

# **全局设计原则**

## **展示原则**

- 异常状态下页面结构不能塌陷。
- 优先保证页面可用，其次再提示异常。
- 单个模块异常不能影响其他模块正常展示。
- 无数据时优先展示空态，不要直接留白。
- 加载失败时优先展示失败提示，不要直接消失。
- 不可点击状态必须有明确视觉区分。
- 所有错误提示文案风格统一，避免同一页面出现多种提示样式。

## **视觉原则**

- 空态：弱化色图标 + 简短说明文案。
- 失败态：浅红或浅灰提示块，不要使用过重警告风格。
- 禁用态：降低透明度、去掉 hover、禁用点击反馈。
- 下架态：卡片灰化 + 遮罩 + “已下架”标识。
- 缺省图：统一默认占位图，不出现浏览器破图图标。

## **文案原则**

建议统一使用以下几类文案：

- 暂无内容
- 加载失败，请稍后重试
- 该内容已下架
- 当前暂无数据
- 当前功能不可用
- 请先登录
- 链接无效

# **页面级异常场景**

## **首页**

### **异常场景**

- 首页弹窗广告为空
- 顶部广告栏为空
- 固定模块无内容
- 一级菜单无内容
- 一级菜单配置异常
- 内容卡片封面缺失

### **UI 要求**

- [ ]  首页弹窗广告为空时，不展示弹窗。
- [ ]  顶部广告栏为空时，整块隐藏，不留空白。
- [ ]  固定模块无内容时，保留模块标题，内容区展示空态，隐藏该模块。
- [ ]  一级菜单无内容时，隐藏该菜单模块。
- [ ]  菜单配置异常时，不展示该菜单模块。
- [ ]  卡片封面缺失时，使用统一默认封面图。

---

## **搜索结果页**

### **异常场景**

- 搜索关键词为空
- 搜索结果为空
- 分类切换后无结果
- 排序失败
- 分页失败

### **UI 要求**

- [ ]  搜索关键词为空时，展示默认提示态。
- [ ]  搜索结果为空时，展示空结果页，保留顶部筛选区。
- [ ]  某个分类无结果时，只替换列表区域为空态，不影响筛选区。
- [ ]  排序失败时，保留当前列表，顶部给轻提示。
- [ ]  分页失败时，保留当前页内容，不跳空白页。

---

## **帖子列表页 / 视频列表页 / 全部分类页**

### **异常场景**

- 列表为空
- 广告为空
- 菜单标题缺失
- 排序参数异常
- 当前页码异常
- 标签筛选为空
- 标签类别过多

### **UI 要求**

- [ ]  列表为空时，展示统一空态。
- [ ]  广告为空时，不展示广告卡片。
- [ ]  菜单标题缺失时，使用默认标题。
- [ ]  排序参数异常时，默认高亮“最新上架”。
- [ ]  页码异常时，默认展示第一页。
- [ ]  标签为空时，隐藏标签区或仅保留“全部”。
- [ ]  标签类别过多时，使用单行横向滚动，不换行。

---

## **动态详情页**

### **异常场景**

- 顶部广告为空
- 正文为空
- 图片为空
- 视频为空
- 上一篇 / 下一篇为空
- 评论为空
- 猜你喜欢为空
- 内容下架

### **UI 要求**

- [ ]  顶部广告为空时，隐藏广告区。
- [ ]  正文为空时，展示“暂无正文内容”。
- [ ]  图片为空时，隐藏图片区域。
- [ ]  视频为空时，隐藏视频区域。
- [ ]  上一篇 / 下一篇为空时，对应按钮置灰或隐藏。
- [ ]  评论为空时，展示“暂无评论”。
- [ ]  猜你喜欢为空时，隐藏该模块或展示空态。
- [ ]  内容下架时，详情页不展示正常正文，统一展示下架提示态。

---

## **视频详情页**

### **异常场景**

- 前置广告为空
- 视频地址失效
- 视频加载失败
- 进度预览截图失败
- 猜你喜欢为空
- 推荐视频为空
- 评论为空
- 内容下架

### **UI 要求**

- [ ]  前置广告为空时，直接展示播放器。
- [ ]  视频加载失败时，播放器区域展示失败态，不黑屏空白。
- [ ]  进度截图失败时，仅展示时间提示，不展示预览图。
- [ ]  猜你喜欢为空时，隐藏模块或展示空态。
- [ ]  推荐视频为空时，隐藏模块或展示空态。
- [ ]  评论为空时，展示“暂无评论”。
- [ ]  内容下架时，播放器不可播放，展示下架提示。

---

## **观看历史页 / 我的点赞页**

### **异常场景**

- 用户未登录
- 当前分类无记录
- 记录为空
- 分页为空
- 历史或点赞数据重复
- 内容已下架

### **UI 要求**

- [ ]  未登录时，优先展示登录引导态。
- [ ]  某个分类无记录时，展示该分类空态。
- [ ]  没有任何记录时，展示整页空态。
- [ ]  分页为空时，隐藏分页器。
- [ ]  下架内容如果仍保留记录，卡片展示“已下架”态。
- [ ]  重复内容只保留一条，不做重复展示。

---

# **组件级异常场景**

## **顶部栏**

### **异常场景**

- 品牌图缺失
- 品牌图加载失败
- APP 下载配置关闭
- APP 下载链接为空
- 登录态异常

### **UI 要求**

- [ ]  品牌图缺失或加载失败时，展示默认品牌图。
- [ ]  APP 下载关闭时，按钮隐藏且其他元素自动补位。
- [ ]  APP 下载链接为空时，建议隐藏按钮。
- [ ]  登录态异常时，默认展示“登录”。

---

## **顶部菜单栏**

### **异常场景**

- 菜单为空
- 菜单名称为空
- 菜单过多
- 子菜单为空
- 跳转地址为空

### **UI 要求**

- [ ]  菜单为空时，仅保留首页和必要兜底菜单。
- [ ]  名称为空时，显示“未命名菜单”。
- [ ]  菜单过多时，单行横向滚动，不换行。
- [ ]  子菜单为空时，不展开或展示“暂无子菜单”。
- [ ]  链接为空时，菜单显示但不可跳转。

---

## **搜索栏**

### **异常场景**

- 输入为空
- 输入超长
- 搜索跳转失败

### **UI 要求**

- 输入为空时，不跳转，不报错。
- 输入超长时，输入框内部滚动，不撑破布局。
- 跳转失败时，保留当前页面状态。

---

## **搜索下拉框**

### **异常场景**

- 历史为空
- 热搜为空
- 删除失败
- 清空失败

### **UI 要求**

- [ ]  历史为空时，历史标题和清空按钮整体隐藏。
- [ ]  热搜为空时，隐藏热搜区。
- [ ]  删除失败时，维持当前数据。
- [ ]  清空失败时，维持当前数据。

---

## **动态内容卡片**

### **异常场景**

- 封面为空
- 标题为空
- 标签为空
- 内容下架
- 跳转链接为空

### **UI 要求**

- [ ]  封面为空时，显示默认封面。
- [ ]  标题为空时，显示“未命名动态”。
- [ ]  标签为空时，隐藏标签区。
- [ ]  内容下架时，卡片灰化并增加“已下架”标识。
- [ ]  链接为空时，卡片不可点击。

---

### **异常场景**

- 封面为空
- 标题为空
- 播放量为空
- 点赞量为空
- 标签为空
- 内容下架
- 跳转链接为空

### **UI 要求**

- [ ]  封面为空时，显示默认封面。
- [ ]  标题为空时，显示“未命名视频”。
- [ ]  播放量或点赞量为空时，显示 0。
- [ ]  标签为空时，隐藏标签区。
- [ ]  内容下架时，卡片灰化并增加“已下架”标识。
- [ ]  链接为空时，卡片不可点击。

---

## **广告内容卡片**

### **异常场景**

- 广告封面为空
- 广告名称为空
- 广告链接为空
- 广告下架

### **UI 要求**

- [ ]  封面为空时，展示默认广告图。
- [ ]  名称为空时，展示“广告位”。
- [ ]  链接为空时，可展示但不可点击。
- [ ]  广告下架时，不展示，也不保留占位。

---

## **评论功能**

### **异常场景**

- 未登录
- 输入为空
- 输入超长
- 提交失败
- 无评论

### **UI 要求**

- [ ]  未登录时，隐藏输入框，显示登录按钮。
- [ ]  输入为空时，评论按钮置灰。
- [ ]  超长时，不允许继续输入或自动截断。
- [ ]  提交失败时，保留输入内容并提示失败。
- [ ]  无评论时，展示“暂无评论”。

---

## **点赞功能**

### **异常场景**

- 未登录
- 接口失败
- 内容下架

### **UI 要求**

- [ ]  未登录点击点赞时，提示登录。
- [ ]  接口失败时，恢复操作前状态。
- [ ]  内容下架时，点赞按钮不可点击。

---

## **分享功能**

### **异常场景**

- 分享链接为空
- 复制失败
- 内容下架

### **UI 要求**

- [ ]  分享链接为空时，按钮可隐藏或点击无效。
- [ ]  复制失败时，提示“复制失败”。
- [ ]  内容下架时，分享功能可隐藏或禁用。

---

## **分页器**

### **异常场景**

- 当前页为空
- 当前页超出范围
- 总数为 0

### **UI 要求**

- [ ]  当前页为空时，默认第 1 页。
- [ ]  当前页超出范围时，回退到最后一页。
- [ ]  总数为 0 时，隐藏分页器。
